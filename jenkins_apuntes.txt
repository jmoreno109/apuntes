
¿Qué es Jenkins?
Servicio de automatización
Código abierto (Java)
Empleado para:
Acelerar el desarrollo de software
(mediante) Automatización del proceso de vida de implementación del proyecto
creación (dado unas plantillas crear un proyecto nuevo listo para desarrollar)
prueba
entrega
Jenkins se puede instalar a través de:

Paquetes del sistema nativo
Docker
Java Runtime Environment (JRE)/WAR
Ventajas
